**Control Hub** — это высокопроизводительная система удаленного администрирования (RMM), состоящая из удобной веб-панели и легковесного агента. Система позволяет удаленно управлять компьютерами на базе Windows прямо из браузера без установки сторонних программ (таких как TeamViewer или AnyDesk).

## 🚀 Основные возможности

* **Высокоскоростная трансляция рабочего стола:** Передача экрана в реальном времени с использованием бинарных WebSockets и аппаратного рендеринга (HTML5 Canvas).
* **Динамическое качество:** Изменение качества потока "на лету" (Низкое/Среднее/Высокое) без разрыва соединения.
* **Pixel-Perfect управление:** Перехват и инъекция событий мыши и клавиатуры напрямую в ядро Windows (с поддержкой любого DPI масштабирования через `MOUSEEVENTF_ABSOLUTE`).
* **Файловый менеджер:** Полноценный браузер файлов для навигации, скачивания и загрузки файлов на удаленный ПК.
* **Удаленный терминал:** Полноценный эмулятор командной строки Windows (PowerShell/CMD).
* **Динамическая сборка агентов:** Веб-сервер "на лету" компилирует уникальный `agent.exe` для каждого скачивания, вшивая в него ваш IP-адрес и секретный токен.
* **Портативность:** Агент представляет собой один единственный исполняемый файл `.exe`, который не требует установки, зависимостей или прав администратора для базовой работы.

## 🛠 Установка на Linux-сервер (VPS/VDS)

Проект полностью упакован в Docker для простого и изолированного развертывания. Серверный контейнер включает в себя как NodeJS, так и компилятор Golang для динамической сборки агентов.

### Требования
* ОС Linux (Ubuntu/Debian/CentOS)
* Установленный `docker` и `docker-compose`
* Открытый порт (по умолчанию 8000)

### Быстрый старт

1. Склонируйте репозиторий:
```bash
git clone https://github.com/TheSuspect1552/ControlHub.git
cd ControlHub
```

2. Выдайте права на запуск скрипта установки и запустите его:
```bash
chmod +x install.sh
./install.sh
```

3. Следуйте инструкциям на экране. Скрипт спросит вас:
   * Какой порт использовать (по умолчанию 8000)
   * Секретный токен (пароль для связи агентов с сервером)
   * Логин и пароль для входа в панель администратора

Скрипт автоматически создаст `.env` файл, соберет Docker-контейнер и запустит панель. База данных SQLite сохраняется в примонтированной папке `data`, поэтому ваши данные не пропадут при перезагрузке сервера.

## 💻 Использование

1. Откройте в браузере вашу панель управления: `http://<IP_ВАШЕГО_СЕРВЕРА>:8000`
2. Авторизуйтесь под логином и паролем, которые вы указали при установке.
3. Перейдите во вкладку **"Настройки" (Settings)**.
4. Нажмите **"Скачать Агента"**. Сервер в реальном времени скомпилирует `agent.exe` специально для вашего сервера.
5. Запустите скачанный `agent.exe` на любом Windows-компьютере (предупреждения Windows Defender или SmartScreen можно проигнорировать, так как приложение не подписано платным сертификатом).
6. Компьютер мгновенно появится в веб-панели со статусом `Online`. Кликните по нему для начала управления!

## ⚙️ Стек технологий

* **Frontend:** React, Vite, Vanilla CSS.
* **Backend:** Node.js, Express, SQLite, WebSockets (`ws`).
* **Agent:** Golang 1.22, `github.com/kbinani/screenshot`, `gorilla/websocket`, Win32 API.
* **Инфраструктура:** Docker, Bash.
